# Artificial_Neural_Networks_Project
UCI Credit Card dataset:
Developed a machine learning system to predict whether a customer will default on credit card
payments using the UCI Credit Card Dataset. Performed data preprocessing including cleaning, feature scaling, and exploratory data analysis(EDA) to uncover patterns and relationships. Implemented multiple classification models such as Logistic Regression, Random Forest, Decision Tree, and Support Vector Machine (SVM) for performance comparison. Evaluated models using classification metrics, confusion matrix, and ROC-AUC curve, achieving reliable prediction performance. Applied PCA and K-Means clustering to visualize data patterns and identify customer segments.

Human Activity Recognition:
Built a machine learning model to classify human activities (e.g., walking, sitting, standing) using
smartphone sensor data. Performed data preprocessing including outlier handling (IQR method), feature scaling (StandardScaler), and label encoding (One-Hot Encoding). Designed and implemented an Artificial Neural Network (ANN) using a Sequential model with multiple hidden layers, ReLU activation, Batch Normalization, and Dropout for improved performance. Trained the model using Adam optimizer with categorical crossentropy loss and applied Early Stopping to prevent overfitting. Evaluated model performance using accuracy, precision, recall, F1-score, and confusion matrix, achieving approximately 94% accuracy on test data.

COVID-19 Data Analysis and Classification:
Developed a data preprocessing and classification pipeline for COVID-19 dataset to improve data
quality and predictive performance.Handled missing and infinite values, detected and treated outliers using the IQR method, and encoded categorical features for modelling. Applied feature scaling and PCA for dimensionality reduction and data visualization. Implemented a KNN classifier to predict WHO regions and evaluated model performance using accuracy and classification metrics.
